3.37 CLASSB_TEST_STATUS

Summary

Identifies result from Class B library test.

Description

This is return type for self-tests.

Remarks

None.

typedef enum 
{
    CLASSB_TEST_NOT_EXECUTED  = 0U,
    CLASSB_TEST_PASSED        = 1U,
    CLASSB_TEST_FAILED        = 2U,
    CLASSB_TEST_INPROGRESS    = 3U
} CLASSB_TEST_STATUS;
